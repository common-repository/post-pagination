== Description ==

Post Pagination plugin is a small yet nice way to increase the default (20) posts per page view in wordpress admin post page.

== Installation ==

1. Unzip/Unrar the plugin folder and copy all the files to YOUR_SITE/wp-content/plugins/ folder.
2. You can also use wordpress plugin uploader to do so.
3. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= What exactly this plugin do ?

Sometime you might need to view more than 20 post in admin post list page, this plugin do exactly the same providing you a select box on the page so admin can choose to show more no of posts.
